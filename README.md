# Proxy Checker Pro

A lightweight desktop application for testing HTTP, HTTPS, SOCKS4, and SOCKS5 proxies.

## Features

- Fast multithreaded proxy checking
- HTTP, HTTPS, SOCKS4 and SOCKS5 support
- Response time measurement
- IP address detection
- Country detection
- Proxy speed estimation
- Export results to CSV
- Simple Tkinter GUI

## Requirements

- Python 3.10+
- requests
- PySocks

## Installation

```bash
git clone https://github.com/dariyaanis/proxy-checker.git
cd proxy-checker
pip install -r requirements.txt
```

## Run

```bash
python proxy_checker_optimized.py
```

## Output

Successful proxies can be exported to CSV files for later use.
